An Open Source website about Open Source projects

Here you can find links to some interesting Free/Open Source Projects that propose nice documentation on how to contribute to them.

This website is an open source project which means that anyone can enrich its content with more links and information.

It contains also some links for other ways of finding interesting FOS projects, and a collection of videos about Free/OpenSoruce contribution.
